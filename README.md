# Next.js SEO Skills

Claude Code skills for implementing and auditing SEO in Next.js App Router projects.

## Installation

Inside Claude Code, run:

```
/plugin marketplace add quochuydev/claude-nextjs-seo-skills
/plugin install nextjs-seo@claude-nextjs-seo-skills
```

## Skills Included

| Skill            | Description                                                                         |
| ---------------- | ----------------------------------------------------------------------------------- |
| `add-nextjs-seo` | Implement SEO: metadata, sitemap, robots.txt, structured data, Open Graph, llms.txt |
| `audit-seo`      | Audit SEO implementation and generate a scored report                               |

## Usage

In Claude Code:

```
/add-nextjs-seo    # Implement SEO for your Next.js project
/audit-seo         # Audit existing SEO implementation
```

## Requirements

- Next.js 13+ with App Router
- Claude Code CLI
